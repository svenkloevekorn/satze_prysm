Was du genau bekommst

Module im Detail

📅
MarketingActivity

Titel, Typ (Social/Newsletter/Launch/PR/Aktion), Datum (oder Zeitraum), Verantwortlich, Status. Die zentrale Einheit.

🔗
Verknüpfungen

Eine Aktivität kann mit FinalProduct, Influencer, Kampagne, Marke verknüpft werden. So sieht man im Produkt: „wann wurde es beworben?".

✅
Freigabe-Workflow

Idee → Entwurf → In Freigabe → Freigegeben → Live → Done. Wer freigeben muss, wer freigegeben hat.

📅
Kalender-View (Filament)

Monats- / Wochen-Ansicht mit farbcodierten Einträgen. Filter pro Typ, Marke, Verantwortlich. Drag & Drop zum Verschieben.

📎
Inhalte-Drafts

Pro Aktivität: Caption, Hashtags, Bilder (Spatie MediaLibrary), Notizen. Endspiel: vollständig brief'bar an externe Tools.

⚠️
Konflikt-Warnung

2 Newsletter am gleichen Tag? 3 Insta-Posts in 2 Stunden? Kalender warnt automatisch.

Warum jetzt

Was es einzigartig macht

1
Verknüpfungen statt Silos

Im FinalProduct-Edit siehst du: „diese 7 Marketing-Aktivitäten sind mit dem Produkt verknüpft". Im Influencer-Profil: „in welchen Posts wurde sie eingesetzt".

2
Eine Wahrheit für alle

Das Marketing-Team, das Produkt-Team, die Geschäftsführung – alle schauen in den gleichen Kalender und sehen das gleiche Bild.

3
Vorausplanung wird einfacher

Welche Wochen sind voll, welche leer? Wo sind Lücken, wo Ballungen? Sofort sichtbar.

4
Archiv für später

„Was hat im Juni 2026 funktioniert?" → ein Filter, fertig. Lerneffekt baut sich automatisch auf.

Realistisch

Aufwand & Phasen

Schritt für Schritt – damit du früh nutzen kannst.

MVP – nutzbar nach
Phase 1 · 1–2 Sessions
Empfohlener Start
  • ✓ MarketingActivity-Resource (Listen-Ansicht + Edit-Form)
  • ✓ 6 Aktivitäts-Typen mit Farben + Icons
  • ✓ Polymorphe Verknüpfungen zu Produkt / Influencer / Marke
  • ✓ Filament-Kalender-Page (Monats-View)
  • ✓ Filter: Typ, Verantwortlich, Marke
Phase 2 · 1 Session
Freigabe-Workflow + Inhalte-Drafts
  • → 6-Status-Workflow (Idee → Entwurf → Freigabe → Live → Done)
  • → Caption / Hashtags / Bilder pro Aktivität
  • → Freigeber zuweisen, „Freigabe anfordern"-Button
  • → Mail-Notification bei „Freigabe nötig"
Phase 3 · 1 Session
Konflikt-Warnung + Performance
  • → Konflikt-Detector: zwei Newsletter am gleichen Tag, mehrere Insta-Posts dicht hintereinander
  • → Verknüpfung mit Performance-Daten (wenn Kampagnen-Manager existiert)
  • → Wochen-View (mit Stunden-Auflösung)
💡 Mein Vorschlag

Phase 1 alleine ist schon der größte Wert: Kalender + Verknüpfungen. Freigabe-Workflow erst dann, wenn du ein Team hast, das ihn braucht – sonst Overhead ohne Nutzen.

Im Vergleich

Marketing-Kalender vs. Kampagnen-Manager?

Beides sinnvoll – aber sie tun verschiedene Dinge:

📅
Marketing-Kalender

Zeit-zentriert: Was passiert wann?

  • • Übersicht über alle Kanäle
  • • Konflikt-Vermeidung
  • • Lange Vorausplanung (3–6 Monate)
  • • Geschäftsführungs-Sicht
🤝
Kampagnen-Manager

Beziehungs-zentriert: Wer macht was für wen?

  • • Influencer-Beziehungs-Management
  • • Briefings & Deliverables
  • • Kosten & ROI pro Influencer
  • • Content-Manager-Sicht
Wenn beide existieren: Kampagnen-Slots erscheinen automatisch im Kalender (verlinkt). Der Kalender ist dann eine Zoom-Out-Sicht über alles – inkl. Kampagnen.
